# Kelime Ezberleme Sistemi

Bu proje, İngilizce kelimeleri 6 sefer tekrar prensibi ile öğrenmek için geliştirilmiş bir web uygulamasıdır.

## Kullanılan Teknolojiler

- Python
- Flask
- SQLite
- HTML
- CSS
- Git / GitHub

## Proje Özellikleri

- Kullanıcı kayıt sistemi
- Kullanıcı giriş sistemi
- Kelime ekleme modülü
- Kelime listeleme ve arama
- Quiz modülü
- 6 tekrar prensibine göre öğrenme sistemi
- Öğrenilen kelimeler sayfası
- Analiz raporu
- Rapor yazdırma özelliği
- Ayarlar sayfası
- Wordle bulmaca modülü
- Word Chain hikaye oluşturucu
- Modern pembe-beyaz arayüz tasarımı

## 6 Tekrar Mantığı

Bir kelime quiz sırasında doğru cevaplandığında tekrar sayısı artar.  
Kelime 6 kez doğru cevaplandığında öğrenilmiş kabul edilir.  
Yanlış cevap verilirse tekrar sayısı sıfırlanır.
